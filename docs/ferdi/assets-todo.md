## 10) Asset Quality Gates (Mandatory)

All newly delivered assets must pass these checks:

1. Must support the locked FERDI palette roles (cream-mineral, smoky blue-gray, restrained navy anchor, warm undertone).
2. Must not introduce dark-SaaS, sci-fi, cyberpunk, neon, or generic glass-heavy direction.
3. Human assets must be integration-ready (blurred-readable), not cut-out portrait style.
4. Motion references must represent slow, low-amplitude atmospheric movement (no flashy transitions).
5. Source files must be included when grading/compositing is part of the asset (to avoid non-editable dead ends).
